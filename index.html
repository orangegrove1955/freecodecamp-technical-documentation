Introduction

Python is a modern, interpreted language, which can be used in many situations to solve problems. The language is particularly useful in the areas of maths and sciences, and has many packages available to expand its capabilities.

Python has many different data types and structures, including (but not limited to) strings, integers, floats, lists, dictionaries and more. Python also has a number of web frameworks, which allow it to be used to create APIs, as well as a large number of machine learning frameworks such as Keras and Tensorflow, which make it an important tool for AI development.

Assumed Knowledge

This documentation asssumes you have basic knowledge of the following:

  • A general understanding of concepts such as variables and data types
  • Some programming experience. If you are new to programming, try one of the tutorials linked in Further Learning
Hello World

As is tradition, we begin in Python by learning how to write a Hello World program

Open a text editor and copy in the following:

def Hello_World: print("Hello World")

This should display the phrase "Hello World" in your terminal when run

Defining Variables

Python is dynamically typed, which means that you don't have to declare what type each variable is. This makes it easy to define a variable, as Python will interpret the type of variable when it is defined. To do so, you can do the following:

  • Define a string: string_var = "My String"
  • Define an integer: int_var = 10
  • Define a float: float_var = 12.19
  • Declare a boolean: bool_var = True
  • Declare a list: list_var = [1,2,3]
Defining Functions

To define a function in Python, simply use the keyword def and give the function a name

def my_function: # Do something here

If...Else Statements

If and Else statements allow you to use truth values to implement conditions

if expression: true statement(s) else: false statement(s)
Loops

There are multiple loop types in Python:

  • For

    For loops have the following syntax:

    for x in list_var: print(x)

  • While

    While loops have the following syntax:

    i = 1 while i < 6: print(i) i += 1
